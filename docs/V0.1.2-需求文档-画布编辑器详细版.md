## 二、顶部导航栏

### 2.1 布局与交互

**布局**：
- 左侧：取消按钮（文字）
- 中间：页面标题"画布编辑"
- 右侧：保存按钮（文字，蓝色高亮）

**交互说明**：
1. **取消按钮**
   - 点击后弹出确认对话框："确定要退出编辑吗？未保存的内容将丢失。"
   - 确认后返回上一页
   - 取消后留在当前页面

2. **保存按钮**
   - 点击后弹出输入框："请输入作品名称"
   - 默认填充当前项目名称（新建时为"未命名作品"）
   - 确认后保存数据到本地存储
   - 保存成功后显示 Toast"保存成功"
   - 1秒后自动跳转到项目列表页

3. **移动端适配**
   - 导航栏顶部增加 `padding-top: var(--status-bar-height)`
   - 按钮点击热区不小于 88rpx × 88rpx
   - 文字按钮增加左右 padding 扩大热区

## 三、顶部工具栏

### 3.1 工具列表

从左到右依次为：

| 图标 | 名称 | 功能说明 |
|------|------|---------|
| ↩️ | 撤销 | 撤销上一步操作 |
| ↪️ | 重做 | 重做下一步操作 |
| 🎯 | 定位 | 回到画布中心/原点 |
| # | 网格 | 打开网格设置弹窗 |
| ↔️ | 镜像 | 切换镜像绘制模式 |
| 📊 | 颜色 | 打开颜色统计弹窗 |
| 🔍 | 大小 | 缩放比例显示与调节 |
| 📑 | 图层 | 打开图层面板 |

### 3.2 各工具详细交互

#### 3.2.1 撤销/重做

**交互说明**：
- 撤销：回退到上一步操作状态
- 重做：前进到下一步操作状态
- 无历史记录时按钮置灰不可点击
- 支持手势：双指左滑撤销，双指右滑重做（可选）

**数据结构**：
```typescript
// 历史记录栈
const historyStack = ref<CanvasData[]>([])
const currentHistoryIndex = ref(-1)

// 每次操作后保存状态
const saveHistory = () => {
  // 删除当前索引之后的历史记录
  historyStack.value = historyStack.value.slice(0, currentHistoryIndex.value + 1)
  // 深拷贝当前状态并压入栈
  historyStack.value.push(JSON.parse(JSON.stringify(canvasData)))
  currentHistoryIndex.value++
}
```

#### 3.2.2 定位

**交互说明**：
- 点击后画布平滑滚动到中心位置
- 如果画布有偏移，恢复到 (0, 0) 偏移
- 如果画布有缩放，恢复到 100% 缩放
- 动画时长：300ms

#### 3.2.3 网格（网格设置弹窗）

**触发方式**：点击顶部工具栏"网格"图标

**弹窗内容**（底部半屏弹窗）：

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 显示网格 | 开关 | 开启 | 控制网格线显示/隐藏 |
| 特殊网格高亮 | 开关 | 开启 | 每 5/10 格加粗显示参考线 |
| 步长(宽/高) | 数字输入 | 10 | 网格格子大小（像素） |
| 起点偏移(X/Y) | 数字输入 | 0, 0 | 网格起始点偏移 |
| 预设尺寸 | 按钮组 | - | 快速选择常用尺寸 |

**预设尺寸按钮**：
- 10×10（小图标）
- 16×16（默认）
- 32×32（中图标）
- 48×48（大图标）
- 64×64（超大图标）

**交互说明**：
1. 弹窗从底部滑出，背景遮罩
2. 点击遮罩或关闭按钮关闭弹窗
3. 修改设置实时预览（不关闭弹窗）
4. 点击"确定"保存设置并关闭弹窗
5. 点击"取消"恢复原始设置并关闭弹窗

**参考截图**：图1（网格设置弹窗）

#### 3.2.4 镜像

**交互说明**：
- 点击后弹出镜像模式选择弹窗
- 选项：
  - 关闭镜像（默认）
  - 水平镜像（以垂直中轴线对称）
  - 垂直镜像（以水平中轴线对称）
  - 四向镜像（水平和垂直同时对称）
- 选择后弹窗关闭，按钮高亮显示当前模式
- 镜像模式下绘制时，自动在对称位置同步绘制

**镜像绘制逻辑**：
```typescript
// 水平镜像
if (mirrorMode.value === 'horizontal' || mirrorMode.value === 'quad') {
  const mirrorX = canvasData.width - 1 - x
  placeBead(mirrorX, y, color)
}
// 垂直镜像
if (mirrorMode.value === 'vertical' || mirrorMode.value === 'quad') {
  const mirrorY = canvasData.height - 1 - y
  placeBead(x, mirrorY, color)
}
```

#### 3.2.5 颜色（颜色统计弹窗）

**触发方式**：点击顶部工具栏"颜色"图标

**弹窗内容**（底部半屏弹窗）：

**顶部统计区域**：
- 颜色数量：当前画布使用的颜色种类数
- 总豆数：当前画布所有拼豆总数
- 库存不足提示（如有）："库存不足，请补充豆子(N)"

**功能按钮**：
- 去除杂点：自动删除孤立的单个拼豆

**智能限色**：
- 滑块调节：减少颜色数量，自动合并相近颜色
- 显示当前限制的颜色数量

**色号列表**：
- 按使用数量倒序排列
- 每行显示：色块、色号、数量、删除按钮、替换按钮
- 库存不足的颜色标记"不足(拥有:0)"
- 点击色号可快速定位到该颜色在画布上的位置（可选）

**交互说明**：
1. 弹窗从底部滑出
2. 实时统计当前画布颜色使用情况
3. 与豆仓管理数据关联，显示库存状态
4. 点击"替换"可将该颜色替换为其他颜色
5. 点击"删除"可将该颜色所有拼豆删除

**参考截图**：图2（颜色统计弹窗）

#### 3.2.6 大小（缩放控制）

**交互说明**：
- 点击后弹出缩放控制弹窗
- 显示当前缩放比例（如 100%）
- 提供滑块调节：25% ~ 400%
- 快捷按钮：25%、50%、100%、200%、400%
- 双指捏合手势也可缩放画布

#### 3.2.7 图层（图层面板）

**触发方式**：点击顶部工具栏"图层"图标

**弹窗内容**（底部半屏弹窗）：

**顶部操作栏**：
- 导入：从本地导入图片作为新图层
- + 新建：创建新的空白图层
- 多选：进入多选模式

**图层列表**：
- 从上到下显示所有图层
- 当前选中图层高亮显示
- 每个图层显示：
  - 缩略图预览
  - 图层名称（可编辑）
  - 可见性开关（眼睛图标）
  - 锁定开关（锁图标）
  - 透明度滑块（0% ~ 100%）
  - 操作按钮：复制、删除

**图层操作**：
- 点击图层：选中该图层
- 长按图层：拖拽排序
- 左滑图层：显示删除按钮

**简化版功能**（V0.1.2 先实现基础功能）：
- 新建图层
- 删除图层
- 显示/隐藏图层
- 图层排序（后续版本）

## 四、画布区域

### 4.1 画布基础功能

**网格显示**：
- 显示网格线（可开关）
- 网格线颜色：浅灰色 #E0E0E0
- 坐标轴标尺（可选）

**画布操作**：
- 单指拖动：平移画布（拖动工具模式下）
- 双指捏合：缩放画布
- 双击：重置画布到 100% 缩放并居中

### 4.2 拼豆绘制

**绘制模式**（由底部工具栏控制）：

| 工具 | 操作 | 效果 |
|------|------|------|
| 画笔 | 点击/拖动 | 放置拼豆 |
| 橡皮擦 | 点击/拖动 | 删除拼豆 |
| 填充 | 点击 | 区域填充相同颜色 |
| 取色器 | 点击 | 吸取该位置颜色 |

**放置拼豆逻辑**：
1. 计算触摸位置对应的格子坐标
2. 边界检查（是否在画布范围内）
3. 放置拼豆（更新 beads 数组）
4. 触发重绘
5. 保存历史记录

### 4.3 触摸事件处理

**事件绑定**：
```html
<view 
  class="canvas-container"
  @touchstart="onTouchStart"
  @touchmove.prevent="onTouchMove"
  @touchend="onTouchEnd"
>
```

**关键处理**：
- `@touchmove.prevent` 阻止画布区域页面滚动
- 使用 `uni.createSelectorQuery()` 获取画布位置
- `requestAnimationFrame` 节流绘制操作

## 六、底部颜色面板

### 6.1 面板结构

从上到下：

**当前颜色区域**：
- 大色块展示当前选中颜色
- 显示色号（如 A01、H07）
- 显示颜色名称（可选）

**最近使用区域**：
- 横向排列最近使用的 8~12 个颜色
- 点击快速切换当前颜色
- 长按可收藏该颜色

**预设色板区域**：
- 按拼豆品牌分类（Artkal、Perler、Hama 等）
- 横向滑动切换品牌
- 每个品牌显示该品牌的标准色号
- 点击颜色选中

**收藏颜色区域**：
- 显示用户收藏的颜色
- 长按颜色可从收藏移除

### 6.2 色板数据

**色号体系**：
- 使用拼豆品牌标准色号
- 如：A01-A200（Artkal）、H01-H100（Hama）等
- 每个色号对应唯一的颜色值

**颜色数据结构**：
```typescript
interface BeadColor {
  code: string      // 色号，如 "A01"
  name: string      // 颜色名称，如 "白色"
  hex: string       // 十六进制颜色值，如 "#FFFFFF"
  brand: string     // 品牌，如 "Artkal"
  category: string  // 分类，如 "基础色"
}
```

## 八、数据保存与还原

### 8.1 保存逻辑

**保存时机**：
- 点击"保存"按钮
- 自动保存（可选，每 30 秒）

**保存数据**：
```typescript
const projectData = {
  id: 'project_' + Date.now(),
  name: projectName,
  canvasData: {
    width: canvasWidth,
    height: canvasHeight,
    beads: JSON.parse(JSON.stringify(beads)), // 深拷贝
    backgroundColor: backgroundColor,
    updatedAt: Date.now(),
  },
  layers: JSON.parse(JSON.stringify(layers)), // 图层数据
  createdAt: Date.now(),
  updatedAt: Date.now(),
  thumbnail: '', // 缩略图（后续生成）
}
```

### 8.2 还原逻辑

**打开项目时**：
1. 从 `pin_projects` 读取项目数据
2. 解析 `canvasData.beads` 数组
3. 还原画布尺寸、背景色
4. 遍历 beads 数组，重新放置拼豆
5. 还原图层数据
